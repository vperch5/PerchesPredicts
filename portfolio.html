Project 1: Basic Python: Entertainment Industry Analysis

Description: In this project, I worked with data from the entertainment industry, focusing on TV shows released during the "Golden Age" of television, beginning in 1999 with the release of The Sopranos. The main objective of the project was to analyze how the number of votes on IMDb impacts the ratings of TV shows. The assumption was that highly-rated shows also tend to have more votes. This project investigates that assumption through data analysis.

Instructions for Completing the Project: The project is divided into three stages:

  • Stage 1: Data Overview — Analyze the dataset and provide insights.
  • Stage 2: Data Preprocessing — Clean the dataset by addressing missing and duplicate values.
  • Stage 3: Data Analysis — Analyze the assumption that shows with high IMDb ratings also have a high number of votes.

Project 2: Instacart Data Analysis

Description: This project involves analyzing a dataset from Instacart, a grocery delivery platform. The dataset contains customer order history, product details, and shopping behaviors. The goal is to clean the data, identify patterns, and generate insights into customer shopping habits.

Instructions for Completing the Project: The project is divided into three main steps:

  • Step 1: Data Overview — Load and examine the dataset.
  • Step 2: Data Preprocessing — Handle missing values, duplicates, and ensure data consistency.
  • Step 3: Exploratory Data Analysis (EDA) — Generate visualizations and insights into customer purchasing behavior.

Key Analyses:

  • Order patterns by time of day and day of the week
  • Customer reorder behaviors
  • Most frequently ordered products
  • Shopping trends based on customer demographics
